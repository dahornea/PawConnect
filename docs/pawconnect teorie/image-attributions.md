# Image Attributions

PawConnect does not currently seed third-party dog photo assets into the database.

For the thesis presentation dataset, seeded dog profiles are intentionally left without stored `DogImage` records unless a valid real image URL is later added through the shelter UI or database. The frontend dog-card and dog-details placeholder remains the fallback for dogs without valid real images.

This avoids storing broken, temporary, placeholder, SVG, cartoon, or unattributed hotlinked image URLs as real dog photos. If real dog photos are added later, record the file name or URL, source, license, and original link in this document.
